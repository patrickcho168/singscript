Singlish Script: Programming with Singlish

If you can speak Singlish, you can be a programmer!

Rules:

  1. Start your expression with ‘bro’ or ‘bang’
  2. End your expression with ‘lah’, ‘leh’, ‘meh’, ‘ah’, ‘lor’
  3. Declare Variables with ‘(var) is (exp)’
    • e.g. bro hacknroll is awesome lah #String
    • e.g. bro boondu is 1 lah #Integer/Float
  4. Print Variables or Strings or Numbers with ‘gimme (exp)’ or ‘simi si (exp)’
    • e.g. bro gimme 3 lah
    • e.g. bro simi si hacknroll leh
  5. If statements follow the following form:
    • if (test) hor then (exp) if not hor (alt)
    • e.g. bro if (goondu > goondus) hor then (gimme goondu) if not hor (gimme goondus) lah
  6. Logical Statements:
    • abuthen = ‘True’
    • talkcocksingsong = ‘False’
    • equal = ‘==‘
    • e.g. bro (7 > 3) equal abuthen ah
    • e.g. bro (3 < 5) equal talkcocksingsoh meh
  7. List Manipulation:
    • Keywords: list, length, seefirst, with
    • e.g. bro hacknroll is (2 with 0 1 6) ah [initialize hacknroll to [2,0,1,6]]
    • e.g. bro abc list meh [check if abc is a list]
    • e.g. bro abc length leh [check the length of abc]
    • e.g. bro abc seefirst leh [see first element of abc]
  8. Functions:
    • fn is (given (vars) (exp))
    • var1 fn (optional vars)
    • e.g. bro print is (given a (gimme a)) lah
    • e.g. bro hacknroll print leh
    • e.g. bro printadd is (given (a b) (gimme (a + b))) lah
    • e.g. bro 1 printadd 3 leh
    • e.g. bro hack printadd roll leh
    • e.g. bro hack printadd (n printadd roll) leh
